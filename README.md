前回の6月はかわいらしい「ストロベリームーン」でしたが、7月の満月は一般的に「バックムーン（Buck Moon／男鹿月）」と呼ばれています。
​名前の由来
​アメリカの先住民が季節を把握するために名付けたもので、「Buck（バック）」とは大人の雄鹿を指します。春に一度落ちた雄鹿の角が、7月頃になると再び立派に生え変わり、大きく成長することに由来しています。

​![レッドクローバー](./%72%65%64%63%6c%6f%76%65%72%2e%70%6e%67)

四つ葉の赤ツメクサ→レッドクローバー　を見つけたら…
もし赤いツメクサで四つ葉（あるいはそれ以上）のものに出会えたなら、それは**「予期せぬ豊かさの到来」や、「あなたの進んでいる道は強力に守られている」**という天からの明確なサインです。
​足元にひっそりと、でも力強く咲くその赤は、「あなたは守られているから、安心して自分のペースで進んで大丈夫だよ」と優しく背中を押してくれているメッセージと言えますね。
​ラズベリームーン（Raspberry Moon）：野生のベリーが実る頃。

​7月29日の夜、もし晴れたら少し夜更かしをして、夏の夜空に輝く「バックムーン」を見上げてみてくださいね。
